# 📊 Super Store Sales Dashboard & Forecast
This project showcases an interactive Power BI dashboard analyzing Super Store sales data, providing insights across various business dimensions including regions, segments, product categories, and a 15-day sales forecast.

## 📈 Dashboard Features
### ✅ Main Dashboard Insights
- **Total Orders**: 22K+
- **Total Sales**: 1.6M
- **Total Profit**: 175K
- **Max Shipment Days**: 4
### 🗂️ Breakdown by:
- **Payment Mode** – COD, Online, Cards
- **Region** – East, West, Central, South
- **Segment** – Consumer, Corporate, Home Office
- **Category & Sub-Category** – Office Supplies, Technology, Furniture
- **Ship Mode** – Standard Class, Second Class, First Class, Same Day
### 🗺️ Geographic Insights
- Sales & Profit by State, visualized on a map
- Top states: California, New York, Texas
## 🔮 Sales Forecast
### 15-Day Forecast
- Line and area charts projecting short-term sales trends
- Patterns identified from Jan 2019 to Jan 2021
- Interactive visuals with zoom/filter options
### 🛠️ Tools Used
- Power BI Desktop
- Excel (data source)
- DAX, Power Query
### 📂 Files Included
- SuperStore Sales Dashboard.pdf – Dashboard snapshots
- SuperStore Sales DataSet.xlsx – Source data
- .png images – Key visuals from the dashboard
